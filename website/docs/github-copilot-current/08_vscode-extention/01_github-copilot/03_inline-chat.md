## インライン補完とチャットの使い分け

### インライン補完機能

- GitHub Copilotがナビゲーター役となり、ペアプログラミングをするような使い方

### チャット機能

- GitHub Copilotに相談・質問をする
- GitHub Copilotにコードレビューをしてもらう
- 細かい指示を行い、コードやドキュメントを生成してもらう
