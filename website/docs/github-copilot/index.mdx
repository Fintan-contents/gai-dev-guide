# GitHub Copilotガイド

## 想定読者と本ガイドの使い方

- 開発プロジェクトにてGitHub Copilotの導入を検討中の方
  - 導入効果・導入の際の考慮事項を把握して、素早く導入検討ができるようになります
    - [GitHub Copilot導入者向け](./github-copilot/for-organization-admin)をお読みください
- 開発プロジェクトにてGitHub Copilotを利用して開発する方
  - すばやく開発環境構築を実施し、効率的な使いこなしができるようになります
    - [インストール&設定](./github-copilot/installation-and-settings)以降をお読みください
